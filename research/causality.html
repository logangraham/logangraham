Causality in Modern Machine Learning: A Review

To be presented at CIRM conference, Nov. 2018. Paper released ~Dec 2018.

In a tweet

ML researchers have turned tremendous interest towards causality. I wrote a review paper detailing the latest research on how tools from causality are improving learning, efficiency, and generalization in machine learning tasks.

Challenge

Causality, among other topics, is one of the quickest growing subfields in machine learning. For various statistical and philosophical reasons, causality has a close relationship to intelligence. Namely, good causal modelling maps to good generative modelling, which is a core challenge in developing generalizable machine learning; further, causal information through interventions are efficient ways of learning desired functional relaitonships. While the machine learning research community has focused mostly on performing causal inference, little attention has been paid to using tools from causality to increase machine learning performance.

Results

I've written a review paper covering 50+ recent papers using tools from causality to improve machine learning. I cover how we can use tools from causality to improve generalization, efficiency, and concept learning. I also define a series of research challenges. See also arXausality, a repo of all arXiv-published causality-in-machine-learning papers in the recent past.
