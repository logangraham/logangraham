What is Pyro?¶

Pyro is a probabilistic programming language (PPL) intended to make it easy to integrate deep learning with probabilistic methods.

You:

  1. Write a graphical model
  2. Define your estimators (deep / not-deep) as part of the model
  3. Choose inference method (MCMC / VI)
  • It's built on top of PyTorch, which makes gradient-based Bayesian deep learning easy
  • Additionally, it appears that its inference primitives make writing your own inference procedures straightforward

In comparison¶

  1. vs. PyMC3 and STAN: same MCMC techniques + variational inference + deep model integration + write intuitively in Python
  2. vs. Edward: dynamic computational graph creation + models are Python functions + intuitive

$$TE = P(Y | do(X=x))$$

Debiasing with Instrumental Variables¶

An instrumental variable $\textbf{Z}$ helps us exploit randomness in treatment to estimate the true treatment effect of $X$ on $Y$. $Z$ is a parent variable of our treatment variable that satisfies 3 criteria:

  1. Relevance: $X \not\!\perp\!\!\!\perp Z$
  2. The Exclusion Restriction: that is, $Z$ affects $Y$ only through the path $Z \rightarrow X \rightarrow Y$, which implies $Z \perp\!\!\!\perp Y | X$.
  3. Moment Restriction / Unconfoundedness: $Z \perp\!\!\!\perp (Y - f(X; \theta_{z}) | X$

In essence, $Z$ provides some randomness of treatment assignment $X$ and we wish to model that randomness "pass-through" to $Y$.

Example: random doctor assignment, treatment number, system outage checks, random service group, random diagnostic model, location proximity changes, rainfall

Standard 2-stage least squares is an interpretation of instrumental variables which assumes $X$ is a linear function of $Z$ and $Y$ is a linear function of $X$, and proceeds as follows:

$$ X \approx Z\beta_{Z} $$$$ \hat{\beta_{Z}} = (Z^{T}Z)^{-1}Z^{T}X $$$$ \hat{X} = Z\hat{\beta_{Z}} $$

and that

$$ Y \approx \hat{X}\beta_{\hat{X}} $$$$ \beta_{\hat{X}} = (\hat{X}^{T}\hat{X})^{-1}\hat{X}^{T}Y $$

Nonlinear instrumental variables model¶

Hartford et al. (2016) extend nonparametric Instrumental Variables to the deep learning case so that we can use arbitrary nonlinear function approximators to capture nonlinearities.

In the standard two-stage scenarios, they show that we can recover the true treatment effect across the space of $X$ if we:

  1. Estimate a first stage model of $P(X | Z)$. This is important that we recover a probabilistic estimate of X as we are going to sample from this posterior.
  2. Perform second stage:
$$\underset{\theta}{\arg\max} \ |D|^{-1}\Sigma_{i}\biggl(y_{i} - \int h_{\theta}(X)\hat{P}_{\phi}(X | Z)\biggr)^{2}$$

This second stage, as we'll see, can also use an arbitrary nonlinear $h_{\theta}$, so long as we draw from the conditional treatment distribution $\hat{P}_{\phi}(X | Z)$.

1. Build the first stage model of:¶

$$\hat{P}_{\phi}(X | Z)$$

For which we're going to use a Gaussian Process (but could easily be any other probabilistic model, such as a mixture density network):

class TreatmentModel(object):
    """
    Class for a general first stage model.
    Any model class can be used, so long as it samples from a posterior P(X_hat | Z, M).
    The model prediction must output a mean and covariance for every Z,
    and sampling from this model must sample from this resulting normal distribution.
    """
    def __init__(self):
        self.hyperopt = True
        self.trained = False
        self.optimizer = torch.optim.Adam
        self.optim_params = {}
        self.n_steps = 1000
        self.verbose = True
        self.print_loss = 50
        self.loss_fn = pyro.infer.Trace_ELBO().differentiable_loss
        
    def _get_model(self, X, y):
        """
        Returns a probabilistic model.
        This can be any probabilistic model, but in this implementation
        returns a Gaussian Process.
        """
        assert y.dim() == 1, "Error: can only handle unidimensional outputs."
        if y.shape[-1] == 1:
            y = y.flatten()
        kernel = gp.kernels.RBF(X.shape[-1],
                                variance=torch.tensor(1.),
                                lengthscale=torch.tensor(1.))
        return gp.models.GPRegression(X=X,
                                      y=y,
                                      kernel=kernel,
                                      jitter=1e-4)
    
    def train(self, Z, X, guide=None):
        """
        Optimize the model using variational inference. Can also
        be optimized with MCMC methods if desired.
        """
        self.pred_dim = X.shape[-1]
        m = self._get_model(Z, X)
        self.trained == True
        pyro.clear_param_store()
        if self.hyperopt:
            if self.verbose:
                print("Beginning hyperparameter optimization.")
            optimizer = self.optimizer(m.parameters(), **self.optim_params)
            self.losses = []
            for step in range(self.n_steps):
                optimizer.zero_grad()
                if guide is None:
                    guide = m.guide
                loss = self.loss_fn(m.model, guide)
                loss.backward()
                optimizer.step()
                self.losses.append(loss.item())
                if self.print_loss is not False and self.verbose:
                    if step % self.print_loss == 0:
                        print("Step {}: {}".format(step, loss))
            self.model = m
            self.guide = guide
        else:
            self.model = m
            self.guide = m.guide
        self.trained = True
        
    def predict(self, Z):
        """
        A predict function that outputs pointwise means and covariances.
        """
        assert self.trained, "Error: train your model first with the `.train(...)` method."
        return self.model(Z)
    
    def sample(self, Z, n=1):
        """
        Sample from P(X_hat | M, Z) where M has been trained on {Z, X}.
        """
        assert self.trained, "Error: train your model first with the `.train(...)` method."
        if Z.shape[0] == 1:
            Z = Z.reshape(-1, 1)
        mu, cov = self.predict(Z)
        K = cov.diag()
        return dist.MultivariateNormal(mu, K).sample((n,)).flatten()
    
    def sample_supervised(self, Z, Y, n=1):
        """
        Samples from P(X_hat | M, Z) and returns vector with corresponding Y vector.
        """
        samples = self.sample(Z, n)
        return torch.cat((samples.reshape(-1, 1),
                          Y.repeat(n).reshape(-1, 1)),
                         dim=1)
